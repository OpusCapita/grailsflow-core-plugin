Release Notes Grailsflow 1.0
-----------------------------

The following new features and improvements were done in release Grailsflow-1.0:

1.  Support of Grails 1.3.x

2.  Support of plain Groovy code in nodes "action" section

3.  Exceptions handling for nodes

4.  Support of users/roles/groups restrictions for Wait nodes execution

5.  Support of forwarding of nodes execution to another users/roles/groups

6.  Support for different views for process variables.

7.  Support translations for Process/Node/Variable

8.  A lot of refactoring and bugs fixes
