Release Notes Grailsflow 0.9
-----------------------------

The following new features and improvements were done in release Grailsflow-0.8:

1. Added some Actions to Grailsflow standard aplication.
   Standard actions: SendMail, SetOwner, Evaluate, Log.
   Remove useless actions.

2. Updated Action Builder

3. Added logging for Grailsflow standard: enhancing sources to write log in specified log file;

4. Fixing Quartz job calls in tests.
